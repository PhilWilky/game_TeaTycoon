## Deliverables Required

### 5 Core Components (All in 9-patch/9-slice format where applicable)

#### 1. Ornate Panel Frame ‚≠ê PRIMARY ASSET

**Format:** Horizontal 9-slice:

  **Style:** Wooden sign/plaque look, slightly darker than frame

#### 3. Buttons (Multiple Sizes)

**Large Button (Primary actions like "RESTOCK"):**

- Normal state
- Hover state (slightly lighter/glow)

- Format: 9-slice or flat PNG with transparency

**Small Buttons (+/- controls):**

- Normal state
- Hover state
- Pressed state
- Size: ~50x50px
- Color: Brown/darker wood
- Format: Flat PNG with transparency

#### 4. Progress Bar Components

**Empty Bar (background):**

- Size: 400x40px
- Style: Inset wooden trough
- Format: 9-slice horizontal

**Fill Bar (foreground):**

- Size: 400x40px (same as background)
- Style: Green liquid/tea filling
- Multiple color variants: Green (stock), Orange (low), Red (depleted)
- Format: 9-slice horizontal

**Optional:** Decorative caps (left/right ends)

#### 5. Background Textures

**Parchment/Paper:**

- Size: 512x512px minimum (tileable)
- Style: Aged paper, subtle texture, warm beige/cream
